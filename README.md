## Инструкция по запуску

Добавить в корень проекта файл .env в который вписать строку VITE_API_TOKEN='ваш_токен'. Токен указан в сданном задании, или его можно получить самостоятельно [на сайте API Поиск кино](https://poiskkino.dev/). Пример также описан в файле .env.default

**Для запуска в режиме разработчика**

В корне проекта выполнить команды:

npm i

npm run dev

(не забудьте добавить в .env токен апи :innocent:)

**Для запуска в режиме build preview**

В корне проекта выполнить команды:

npm i

npm run build

npm run preview

(не забудьте добавить в .env токен апи :innocent:)

## Описание приложения
**Стек**: Приложение реализовано на React и Typescript. Роутинг сделан на React Router. Запросы выполняются с помощью Axios. В качестве библиотеки компонентов использована MaterialUI. Сборка произведена на Vite. Реализованы тосты для ошибок с использованием react-tostify.

Приложение отображает список фильмов, полученных с помощью [API Поиск кино](https://poiskkino.dev/). На главной странице реализован список полученных фильмов с **бесконечным скроллом и фильтрацией по жанрам, году и рейтингу**. Состояние фильтров **синхронизируется с SearchParams**, засчет чего можно отправлять ссылку на конкретные значения фильтров на странице. При нажатии на карточку фильма производится **переход на отдельную страницу с информацией о нем**.

В хэдере сайта реализована навигация по страницам - главная и избранное. Также реализована **поисковая строка для поиска по названию** - при вводе строки ищутся фильмы с соответствующим названием. При нажатии на строку фильма производится переход на страницу фильма. В поиске реализован **дебаунс на 500мс** с целью минимизации лишних запросов API. 

Реализован **раздел Избранное**. Добавить и удалить фильм из избранного можно на отдельной странице фильма, с использованием **модального окна для подтверждения**. Для списка фильмов в избранном реализована отдельная страница, а **хранение фильмов производится в LocalStorage** для сохранения данных при перезагрузке страницы.

На Главной странице в левой верхней части экрана находится **Режим сравнения фильмов** - при нажатии на иконку откроется сайдбар со сравнительной областью. Сайдбар также откроется автоматически при добавлении фильма в режим сравнения (с помощью кнопки на карточке фильма, или кнопки в списке фильмов поиска по названию). При добавлении третьего фильма **первый автоматически сбрасывается**. Режим сравнения реализован с использованием **React Context**, чтобы добавлять фильмы можно было из разных мест в приложении без излишнего проп-дриллинга.

В приложении присутствует основная (светлая) тема и **темная тема**, которые можно переключить с помощью иконки в правой части хэдера. Иконки на сайте подписаны тултипами для удобства :wink:
